QUE ES GIT

Git es un sistema de control de versiones distribuido de código abierto y gratuito diseñado para manejar todo, desde proyectos pequeños a muy grandes, con velocidad y eficiencia. Git es fácil de aprender y ocupa poco espacio con un rendimiento increíblemente rápido . Supera a las herramientas SCM como Subversion, CVS, Perforce y ClearCase con características como bifurcaciones locales económicas , áreas de preparación convenientes y múltiples flujos de trabajo .

El comando git pull se emplea para extraer y descargar contenido desde un repositorio remoto y actualizar al instante el repositorio local para reflejar ese contenido. El comando git pull es, en realidad, una combinación de dos comandos, git fetch seguido de git merge.

El comando git push se usa para cargar contenido del repositorio local a un repositorio remoto. ... Es el equivalente de git fetch , pero en lugar de descargar importaciones de commits en ramas locales, envía exportaciones de commits a ramas remotas.
